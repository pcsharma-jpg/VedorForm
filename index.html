Apps Script Setup Guide
1
Google Sheet banao
Google Drive → New → Google Sheets → naam: "ParallelDots Vendor Onboarding". Sheet URL se ID copy karo (/d/XXXXX/edit mein se XXXXX).
2
Drive folder banao
Google Drive → New Folder → "Vendor Documents". Folder open karo, URL se ID copy karo (/folders/XXXXX mein se).
3
Apps Script mein code paste karo
Sheet → Extensions → Apps Script → Code.gs mein neeche wala code paste karo. SHEET_ID aur DRIVE_FOLDER_ID update karo.
4
Deploy karo
Deploy → New Deployment → Web App → Execute as: Me → Who has access: Anyone → Deploy → URL copy karo.
5
URL yahan paste karo + vendor forms mein bhi
Upar wale field mein URL daalo. Vendor HTML files mein bhi APPS_SCRIPT_URL replace karo.
